# MarkTab 隐私政策

最后更新：2026-05-19

MarkTab 是一个用于在 Chrome 新标签页展示、搜索和整理本地浏览器书签的扩展。

## 数据使用

扩展会读取浏览器书签，用于在新标签页和弹出窗口中展示书签、文件夹数量、分类列表和搜索结果。

扩展会保存少量设置数据，包括隐藏的分类 ID、置顶书签 URL、主题、侧边分类栏折叠状态。这些设置通过 Chrome Storage 保存，用于在同一浏览器或 Chrome 同步环境中保留用户偏好。

扩展会使用 Chrome 内置 favicon 服务显示书签对应网站图标。图标加载失败时，界面会回退到首字母图标。

## 数据传输

扩展不会把书签标题、书签 URL、分类名称或设置数据发送到开发者服务器。

扩展不会出售、出租或共享用户数据给广告平台、数据经纪商或其他第三方。

用户主动在搜索框中提交网页搜索时，扩展会通过 Chrome Search API 使用用户在 Chrome 中设置的默认搜索引擎。这是搜索功能的预期行为。

## 远程代码

扩展不加载或执行远程托管代码。扩展的 HTML、CSS、JavaScript 和图标资源都随扩展包一起提供。

## 权限说明

- `bookmarks`：读取书签树并创建用户主动添加的书签。
- `favicon`：使用 Chrome 内置 favicon 服务显示书签对应网站图标。
- `search`：在用户主动提交网页搜索时，通过 Chrome Search API 使用浏览器默认搜索引擎。
- `storage`：保存用户设置，例如隐藏分类、置顶书签和主题。
- `activeTab`：在用户点击扩展弹出窗口中的“添加当前页面”时读取当前标签页信息，用于创建书签。
